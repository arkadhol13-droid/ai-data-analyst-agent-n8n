# AI Data Analyst Agent (n8n) 🤖📊

This project is an AI-powered Data Analyst Agent built using n8n automation platform.  
The agent analyzes datasets, generates insights, and can automatically send analysis reports via email.

---

## 📌 Project Overview

The AI agent simulates the workflow of a professional data analyst by performing automated dataset analysis including:

- Data overview
- Data quality checks
- Trend detection
- Business insights
- Visualization recommendations
- Actionable recommendations

The workflow integrates AI models with automation tools to create a smart data analysis assistant.

---

## 🛠 Tools & Technologies

- n8n (Workflow Automation)
- Groq LLM
- Google Sheets
- Gmail Automation
- AI Agent (LangChain Nodes)

---

## ⚙ Workflow Architecture

The workflow includes the following nodes:

1️⃣ Chat Trigger  
Receives user input from chat interface.

2️⃣ AI Agent  
Processes the dataset and performs analysis using LLM.

3️⃣ Memory Buffer  
Maintains context for conversation.

4️⃣ Groq Chat Model  
Runs the AI language model for analysis.

5️⃣ Google Sheets Tool  
Fetches dataset from Google Sheets.

6️⃣ Gmail Tool  
Automatically sends the generated analysis report via email.
